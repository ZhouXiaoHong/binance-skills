# Cursor Rules Generator

A skill for generating Cursor IDE rules and agent configurations from Handlebars templates.

## Installation

```bash
npx bnskills add cursor-rules-generator
```

## Templates

- `rule.hbs` - Generate `.cursor/rules/*.md` files
- `agent.hbs` - Generate agent configuration files

## Note

This README is for human reference and will be excluded during skill installation. The AI agent uses `SKILL.md` as its entry point.
